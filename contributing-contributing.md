First of all, thank you for considering a contribution to Firefly III. I welcome all help!

## Translations

Both [Firefly III](https://crowdin.com/project/firefly-iii) and the [Firefly III help pages](https://crowdin.com/project/firefly-iii-help) are
on CrowdIn, ready to be translated in variety of languages. Is your language not listed? Send me a message and I will add it!

## Bugs

If you can fix a bug, you are more than welcome to create a pull request against the `develop`-branch with your code. Many users have done so.

## Features

If you want to add something to Firefly III, you're also more than welcome to open a pull request!
